# GO VO

执行上下文入栈后创建变量对象，用于存储一些信息。

函数执行上下文的变量对象又称为`VO`

```js
let obj = {
    name: 'John',
    fn: (function(x){
        return x + 10；
    })(obj.name);
}//执行会报错。
```

需要弄清楚变量创建的过程。

1. 在`Heap`中创建值
2. 创建变量
3. 关联在一起
